# Tetris - in Assembly Language (MARS MIPS)

## Bitmap Display Parameters:
- Unit (Width : Height) in Pixels:
    - (4 : 4);
    - (2 : 2); or
    - (1 : 1).
      
- Display (Width : Height) in Pixels:
    - (512 : 1024) for (4 : 4);
    - (256 : 512) for (2 : 2); or
    - (128 : 256) for (1 : 1).
      
- Base address for display:
    - 0x10040000 (heap)
